Introduction to HTML

History, Uses, Basic Structure, and Deployment

Presented by: Software Engineer

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It structures the content on the web and allows for the embedding of images, text, links, videos, and more.

  • Developed by Tim Berners-Lee in 1991
  • Markup language, not a programming language
  • Uses elements and tags to format content

History of HTML

  • 1991: HTML 1.0 - Basic text formatting and hyperlinks.
  • 1995: HTML 2.0 - Forms and tables added.
  • 1997: HTML 3.2 - Introduction of CSS, JavaScript support.
  • 1999: HTML 4.01 - Accessibility improvements and separation of content/design.
  • 2014: HTML5 - Multimedia support, semantic tags, and APIs.

Why Use HTML?

  • Core language for web development.
  • Easy to learn and widely supported.
  • Works with CSS and JavaScript to create dynamic and responsive websites.
  • Free and open standard across all browsers.

Basic Structure of an HTML Document

<!DOCTYPE html>
<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
    <p>This is my first web page.</p>
  </body>
</html>

The structure includes:

  • <!DOCTYPE html>: Declares the document type.
  • <html>: Root element of the document.
  • <head>: Contains meta information like title.
  • <body>: Contains the visible content of the page.

Key HTML Tags

  • <h1> to <h6>: Headings.
  • <p>: Paragraphs.
  • <a>: Hyperlinks.
  • <img>: Images.
  • <div> and <span>: Layout containers.
